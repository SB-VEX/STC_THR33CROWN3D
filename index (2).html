Nationals · Robot Skills Autonomous
84
cactus · fully autonomous

A fully programmed autonomous routine using PID control with the Brain inertial sensor — engineered from scratch. No driver input.

PID controllerInertial sensorKp Ki Kd tuned
# PID turn — core of the autonomous
def turn_pid(target):
Kp = 1.5 # proportional
Ki = 0.002 # integral
Kd = 3.0 # derivative
while abs(error) > 1:
error = target - inertial.heading()
output = Kp*error + Ki*integral + Kd*deriv
set_drive(output, -output)
